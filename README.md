# Snapdocs Email Rewriter

This tool standardizes transactional emails across signer, lender, settlement, and notary audiences.

## Features

- Extracts content from HTML emails
- Rewrites emails using structured templates
- Enforces email standards (CTA, subject lines, structure)
- Generates:
  - Preview
  - HTML
  - HAML
  - JSON

## How to run locally

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
cd backend  
npm install  
npm run dev  

## Standards

- email-template-standards.md
- copy-standards.md

## Usage

Use the Email Rewriter to:
- create new emails
- standardize existing emails
- generate engineering-ready templates
